Entwicklung und Support¶

Lizenz¶

QField ist unter der GNU Public Licence (GPL) Version 2 oder später verfügbar. Das heisst, dass der Quellcode immer eingesehen und geändert werden kann und garantiert, dass es immer ein auf QGIS basierende App zum Erheben von Daten im Feld geben wird, das Gratis ist und frei angepasst werden kann.

Der Quellcode liegt auf github

Nennungen¶

QField wird von OPENGIS.ch entwickelt. OPENGIS.ch bietet Beratung, Entwicklung und Schulungen für Open Source Software inklusive QField und QGIS an.

Lass es wachsen, mach es Deins¶

Dies ist eine Liste mit Features, die unserer Meinung nach wichtig und nützlich sind für einen effizienten Arbeitsablauf.

  • Polygone digitalisieren

  • Zusätzliche Editier-Widgets

  • Automatische Synchronisation basierend auf der Netzwerkverbindung

  • Verbesserte Koordinatenpräzision

  • Geometrien editieren (Kann in mehreren Schritten entwickelt werden)

  • GPS Status Informationen in Attribute speichern während dem Digitalisieren (Präzision, RTK Status ...)

  • Search functionality

  • Anschliessen von externen Messgeräten
