Xestión do Proxecto QField¶

Precisarás unha versión portable do ficheiro de proxecto QGIS (.qgs). Por portable enténdese que tódalas rutas son relativas e os conxuntos de datos están accesibles dende o dispositivo.

Comproba que Proxecto ‣ Propiedades de Proxecto ‣ Xeral ‣ Gardar rutas está establecida a “Relativa” e que tódolos ficheiros de datos solicitados están no mesmo cartafol que o ficheiro .qgs ou nun sub-cartafol del.

Para incrementar a produtividade, suxerimos botar unha vista ós seguintes plugins:

  • Plugin QConsolidate

  • Plugin de Edición sen Conexión

Coordinate Reference Systems¶

QField uses the Coordinate Reference System (CRS) information found in the project file.

  • Project CRS
  • On the fly reprojection (OTF) is used for visualization
  • Layer CRS

Note

QField does not respect OTF when digitizing new features. Make sure that all the layers on which you digitize are in the same CRS like the project. If you fail to do so it will look as if new features are not saved because they end up in far far away.

Configuración de Capa Vectorial¶

A maioría dos axustes que ofrece QGIS son admitidos directamente por QField sen ningún esforzo extra.

Estilo¶

Toda a configuración de estilo de QGIS admítese directamente por QField. Isto inclúe tódolos tipos de renderización como graduado, por categorías, baseado en normas, 2.5D así como a simboloxía definida polos datos.

Formularios e Campos¶

QField crea formularios de forma similar, pero non igual a QGIS.

Eliminar Formulario de Entidade

A configuración suprimir formulario de entidade aplícase directamente ó formulario de QField.

Lembrar os últimos valores

QField ofrece un control máis refinado sobre os últimos valores usados e ignora a configuración de QGIS.

Configuración e símbolos SVG personalizados¶

No diálogo ‘acerca de’ podes ver onde están os cartafoles compartidos no dispositivo.

Se queres utilizar símbolos personalizados tes que poñelos alí empregando un xestor de ficheiros.

External SD card¶

In order to have write access to projects which are on an external SD card, you will have to put it into the following folder:

/Android/data/ch.opengis.qfield/files
