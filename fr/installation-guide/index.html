Guide d’installation QField¶

Version Android¶

QField peut être utilisé à partir de la version 4.3 d’Android.

Télécharger à partir de GooglePlay¶

Vous pouvez télécharger l’application directement depuis GooglePlay

Obtenir un apk pré-buildé¶

Il est possible de télécharger un apk pré-buildé pour ceux qui ne peuvent pas ou ne souhaitent pas utiliser Google Playstore. Télécharger qfield.apk.

Edition Karma¶

Avertissement

Il ne s’agit pas de l’app QField. C’est un moyen de participer financièrement au développement de QField.

Si vous avez acheté la version de l’application `QField Karma edition<https://play.google.com/store/apps/details?id=ch.opengis.qfieldkarmaedition>`_, nous vous remercions pour votre participation. Pour obtenir QField, il faudra l’installer selon les instruction ci-dessus.
